🚀 AI Lead Classifier MVP.

A simple full-stack MVP that captures user leads and uses AI to automatically classify them into actionable categories.

🌐 Live Demo

👉 https://ai-lead-classifier-e18o.onrender.com

🎯 Overview

This project was built as a rapid prototype to demonstrate how AI can be integrated into real-world workflows to automate decision-making and reduce manual effort.

The system allows users to submit their information through a form, processes the input using an AI model, and classifies each lead into categories such as Sales, Support, or Other.

⚡ Key Features
Lead capture form (name, email, message)
AI-powered classification using OpenAI
SQLite database for storage
Simple dashboard to visualize submissions
Fast MVP built with focus on execution speed
🧠 AI Integration

Each submitted message is sent to the OpenAI API and classified into:

Sales
Support
Other

This simulates how businesses can automate lead routing and prioritization.

🛠️ Tech Stack
Backend: Node.js (Express)
Frontend: HTML + Tailwind CSS
Database: SQLite
AI: OpenAI API
Tools: Axios, dotenv

⚠️ Notes
This is a rapid MVP focused on functionality over perfection.
AI outputs are validated but may require tuning for production use.
Environment variables are required to run the project.
